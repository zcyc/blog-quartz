# 总结

所有框架都有自己独特的组件库，而且互不通用。
经过多年的发展，有些企业开源了自家设计体系，才让不同框架拥有了风格统一的组件。
谷歌设计体系覆盖了所有前端框架。
蚂蚁设计体系目前只涵盖了三大框架。
头条的设计体系只覆盖了 Vue 和 React。
腾讯设计体系覆盖了三大框架，但是都没开发完。
CSS 样式和基于 Tailwind 的组件都适用于所有框架，但是没有 JS 组件，比如国内常用的日期时间选择器。
Bootstrap 不是纯 CSS 组件，但支持所有框架。
